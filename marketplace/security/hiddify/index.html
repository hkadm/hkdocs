    Hiddify¶

    Информация

    Hiddify - это панель с широким спектром функций и возможностей, позволяющая легко и безопасно выходить в интернет. Hiddify позволяет скрыть активность в сети от злоумышленников. и предотвращает типовые атаки на сервер, обеспечивая надежное шифрование трафика по алгоритму AES-256.

    Начало работы после развертывания Hiddify¶

    После оплаты заказа на указанную при регистрации электронную почту придет уведомление о готовности сервера к работе. В нем будет указан IP-адрес VPS, а также логин и пароль для подключения. Управление оборудованием клиенты нашей компании осуществляют в панели управления серверами и API — Invapi.

    Данные для авторизации, которые можно найти или во вкладке Info >> Tags панели управления сервером или в присланном e-mail:

    • Логин и Пароль для управления сервером: приходят в письме на вашу электронную почту при готовности сервера к работе после развертывания ПО.

    Примечание

    Подробная информация по основным настройкам Hiddify содержится в документации разработчиков.

    Подключение к серверу и вход в систему Hiddify¶

    Для подключение к серверу и вход в систему Hiddify следует выполнить следующие шаги:

    1. Открыть терминал и подключиться к серверу с помощью команды:

      ssh root@<server_ip>

    2. При первом подключении необходимо подтвердить аутентичность сервера - ввести "yes", чтобы добавить ключ в список известных хостов.

    3. После успешной авторизации на экран терминала будет выведено стандартное уведомление безопасности. Необходимо нажать кнопку Ok

    4. В следующем окне выбрать пункт "Show admin link":

    5. На экран терминала будут выведены данные для доступа к административной панели Hiddify:

    6. В панели Hiddify выбрать страну Others.

    7. В поле Domain указать домен вида hiddify{id сервера}.hostkey.in, например, hiddify48543.hostkey.in.

      Примечание

      Найти ID и IP сервера можно в панели управления

    8. В поле Decoy Site вы можете задать URL сайта, который будет выводиться при заходе по IP-адресу или доменному имени из п.3.

    9. Нажать кнопку Submit:

    10. Установка занимает примерно пять минут. Обязательно скопируйте и сохраните ссылку доступа к панели от администратора вида https://48543.hostkey.in/oN6xxKuCjrLIm0Garp2SGtMgH/0d3603c3-a8a3-47ad-9ee1-6c56044aae14/admin/, которая будет показана на этом этапе.

      Внимание

      Если вы не сохранили ссылку администратора, то вы можете восстановить ее.

    11. При успешной установке вы получите сообщение:

      Внимание

      Заходить в панель администратора нужно только по ссылке, которую вы сохранили в п.6, так как после установки по адресу {id сервера}.hostkey.in вам будет показываться случайный сайт или сайт, заданный в п.4.

    Добавление нового пользователя¶

    Управление пользователями осуществляется в разделе Users.

    Примечание

    После установки вам сразу доступен пользователь default.

    Для создания нового пользователя необходимо перейти в указанный раздел и нажать на кнопку Create:

    Затем заполнить данные о новом пользователе. Строка Name является обязательной для заполнения:

    Поля Usage limit позволяет установить объем трафика для пользователя, а строка Package Days позволяет установить количество дней, на протяжении которых пользователь может использовать этот объем трафика.

    После внесения данных о пользователе необходимо нажать на кнопку Save.

    Панель управления пользователя¶

    Чтобы перейти в панель управления конкретного пользователя из панели администратора, нужно нажать на кнопку адреса в столбце User Link в строке данного пользователя:

    В открывшейся панели можно:

    • Посмотреть оставшийся трафик в поле Traffic Usage нажав на Display Usage Details;

    • Получить QR-код или ссылку для настройки клиентов доступа к Hiddify, нажав Display QR Code в поле QR Code;

      В этом случае откроется QR-код, который можно считать камерой смартфона или, нажав на кнопку Copy, скопировать ссылку доступа для сторонних клиентов.

      Информация

      По умолчанию выбирает конфигурация AUTO. Для выбора конкретного протокола вам нужно зайти во вкладку All Configs панели управления пользователя.

    • Настроить уже установленный в системе клиент HiddifyN, нажав на поле Tap to set Anti-Filter;

    • Получить короткую ссылку для доступа к панели управления пользователя, нажав на поле Temporary Short Link.

    Подключение к Hiddify с ПК¶

    Использование клиента HiddifyN¶

    Примечание

    Ссылка на инструкцию разработчиков по работе с клиентом HiddifyN.

    HiddifyN - программное обеспечение на основе Xray. Для использования этого клиента необходимо:

    1. Скачать программу. Для этого во вкладке Desktop Hiddify открыть раздел, связанный с HiddifyN, затем нажать на кнопку Windows.

      Примечание

      Для Linux и macOS выбирайте клиент HiddifyClash.

    2. Установить SDK, а затем извлечь и запустите загруженный файл HiddifyN в соответствующей папке.

    3. Добавить профиль нового пользователя в приложение. Для этого необходимо на сервере перейти в панель управления Hiddify и нажать кнопку Tap to import. Это действие автоматически добавит конфигурацию подключения в клиент на ПК.

    4. Выбрать режим прокси.

      Одна из важнейших частей HiddifyN - режимы прокси:

      • Auto (Авто). Автоматическое подключение к конфигурации с наивысшей скоростью (наименьшим пингом).
      • Load Balance (Баланс нагрузки). Одновременное подключение к нескольким конфигурациям (трафик распределяется между ними). Этот режим полезен для распределения нагрузки на несколько конфигураций одновременно для достижения приемлемой скорости.
      • Manual (Вручную). Используется для ручной настройки конфигураций. Этот режим позволяет выполнить расширенные настройки программы.

    5. Подключиться к серверу приватной сетевой инфраструктуры - нажать на среднюю круглую кнопку. Программное обеспечение подключается к конфигурациям на основе выбранного режима прокси, а рядом с этой кнопкой отображается пинг, связанный с конфигурацией:

    Использование клиента Hiddify Next под Windows, Linux или macOS¶

    Для подключения к Hiddify с помощью клиента Hiddify Next необходимо:

    1. Скачать релиз с тегом Latest и установить установочный пакет или файл под вашу операционную систему.

    2. В Проводнике перейти по пути установки (обычно C:\Program Files\hiddify), из меню, вызываемом по нажатию правой кнопкой мыши на файле HiddifyNext.exe выбрать Свойства и в открывшемся окне перейти на вкладку Совместимость.

      Далее нажать на кнопку Изменить параметры для всех пользователей и отметить чекбокс Запускать эту программу от имени администратора и нажать на OK.

    3. Запустить клиент Hiddify Next и выполнить следующие настройки:

      • В разделе Настройки выбрать Регион - Россия (ru) если вы будете использовать клиент в РФ или Другой.

      • В разделе Настройки >> Параметры конфигурации установить Режим работы отличный от Прокси и Системный прокси (как показано на скриншоте ниже):

    4. Добавить данные сервера. Для этого:

      • Получить ссылку для настройки доступа к Hiddify, нажав Display QR Code в поле QR Code панели управления пользователя
      • В открывшемся окне с QR-кодо нажать на кнопку Copy, скопировать ссылку доступа для сторонних клиентов.

      • Перейти в раздел Главная и добавить подключение к серверу, нажав на кнопку + Новый профиль или на значок + в правом верхнем углу
      • Нажать на Добавить из буфера обмена:

      При успешном добавлении данных появится всплывающее сообщение и профиль сервера появится в списке:

    Для подключения к Hiddify вам нужно нажать на большую круглую кнопку. Если соединение будет установлено, значок на ней станет зеленым и появится надпись Подключено. Отключение производится аналогично кликом по кнопке.

    Мобильные устройства Android¶

    Подключение с помощью HiddifyN¶

    Для подключения к Hiddify с мобильных устройств Android также возможно использовать клиент HiddifyN. После перехода по ссылке из панели управления Hiddify возможно либо скачать клиент напрямую по ссылке, либо использовать Google Play.

    После установки приложения необходимо выбрать язык и страну. Затем скачать ссылку для подключения и нажать на кнопку Click to connect:

    Для подключения к клиенту HiddifyN также возможно использовать QR-код. Для этого необходимо кликнуть на кнопку Android, затем на значок рядом с надписью Tap to import, а затем сканировать QR-код из приложения на смартфоне:

    Если возникли проблемы со сканированием QR-кода, то можно использовать скопировать ссылку, нажав на Copy и затем сохранить ее для передачи на смартфон. Или, если вы откроете панель управления клиента на смартфоне, вы можете открыть ссылку напрямую на нем, нажав на Tap to import.

    Подключение с помощью Hiddify Next¶

    Для подключения к Hiddify с помощью Hiddify Next необходимо:

    1. Скачать и установить Hiddify Next из Google Play или apk под вашу архитектуру с сайта разработчика.

    2. Открыть приложение Hiddify Next и выполнить предварительные настройки:

      • Зайдя в пункт меню >> Настройки выбрать Регион - Россия (ru) если вы будете использовать клиент в РФ или Другой;
      • В меню >> Настройки >> Параметры конфигурации установить Режим работы, отличный от Прокси, как показано на скриншоте ниже:

    3. Добавить данные сервера. Для этого:

      • Получить QR-код с ссылкой для настройки доступа к Hiddify, нажав Display QR Code в поле QR Code панели управления пользователя:

      • В приложении Hiddify Next перейти на основной экран и добавить подключение к серверу, нажав на кнопку + Новый профиль или на значок + в правом верхнем углу, после этого нажать на Сканировать QR-код и считать его:

    4. При успешном сканировании будет выведено сообщение и профиль сервера появится в списке. Для подключения к 3X-UI вам нужно нажать на большую круглую кнопку. Если соединение будет установлено, значок на ней станет зеленым и появится надпись Подключено. Отключение производится аналогично кликом по кнопке.

    5. Вы можете выбрать, для каких приложений вам нужно включать (или не включать) прокси-режим. Для этого перейдите в меню >> Настройки и установите переключатель Раздельное проксирование в ВКЛ. Затем, нажав на значок рядом с ним, установите чекбокс режима проксирования и выберите нужные вам приложения.

    Обновление Hiddify¶

    Для обновления панели вам необходимо зайти в раздел Actions и выбрать пункт Update. Примерно через пять минут система будет переустановлена.

    !(./images/hiddify_0_9.png)

    Внимание

    Не выключайте питание и не перезагружайте сервер во время этого процесса. Обновление может не произойти, если оно сломано у разработчиков.

    Восстановление ссылки администратора через консоль¶

    Для того, чтобы получить ссылку администратора, проделайте следующие шаги:

    1. Зайдите на сервер по ssh из терминала или HTML5 консоли:

      ssh root@{IP-адрес вашего сервера}

    2. Запустите скрипт (скопируйте в консоль и нажмите Enter)

      /opt/hiddify-manager/menu.sh

    3. Выберите пункт меню admin Show admin link и нажмите Enter:

    4. Выделите и скопируйте ссылку клавишами Ctrl+C или Shift+Ins

    Hiddify. Основные возможности¶

    • Быстрая установка;
    • Несколько ядер Xray и SingBox;
    • Умный прокси (клиенты Hiddify и Clash);
    • Автоматическое обновление и резервное копирование;
    • Автоматическое подключение к Cloudflare;
    • Использование множественных доменов;
    • Возможность автоматической настройки CDN IP;
    • Управление активными конфигурациями;
    • Многоуровневые права администратора;
    • Единая панель SingBox с функцией управления пользователями;
    • Ограничения по времени и трафику для каждого пользователя;
    • Telegram-прокси с использованием разных библиотек;
    • DNS через HTTPS (DoH);
    • Управление пользователями через Telegram бота.

    Особенности сборки¶

    • Возможна установка на Ubuntu 20.04 и 22.04. Генерацией сертификатов на домены занимается панель;
    • Установка панели Hiddify без докера в /opt;
    • Установленное ПО: - hiddify/hiddify-config;
    • Путь до приложения: - /opt/hiddify-config.
    • Веб-интерфейс будет доступен после установки в личном кабинете по уникальной ссылке.
