    Информация

    H-UI VPN сервер - это графический пользовательский интерфейс (GUI) для управления сервером и клиентом Hysteria — современного, высокопроизводительного протокола обхода цензуры и прокси-трафика, разработанного как альтернатива Shadowsocks, V2Ray и другим.

    Начало работы после развертывания H-UI VPN сервер¶

    После оплаты заказа на указанную при регистрации электронную почту придет уведомление о готовности сервера к работе. В нем будет указан IP-адрес VPS, а также логин и пароль для подключения. Управление оборудованием клиенты нашей компании осуществляют в панели управления серверами и API — Invapi.

    Данные для авторизации, которые можно найти или во вкладке Info >> Tags панели управления сервером или в присланном e-mail:

    • Ссылка для доступа к панели управления H-UI VPN сервер: в теге webpanel. Точная ссылка вида https://hui{Server_ID_from_Invapi}.hostkey.in приходит в письме, отправляемом при сдаче сервера;
    • Логин: useradmin;
    • Пароль: приходит в письме на вашу электронную почту при сдаче сервера.

    Примечание

    Подробная информация по основным настройкам H-UI содержится в документации разработчиков.

    Особенности сборки¶

    ОС: Ubuntu Минимальные системные требования: 1 vCPU, 1 GB RAM, 15 GB SSD

    Информация

    Hysteria 2 не требует TLS-сертификата, но использование домена делает трафик похожим на обычный HTTPS (обход DPI).

    Настройка конфигурации прокси-сервера Hysteria2¶

    Перед началом использования необходимо настроить прокси-сервер Hysteria2. Перейдите во вкладку Hysteria Manage. Вы увидите уведомление Hysteria2 Status:Stop

    Заполните параметры перед запуском: 1. На вкладке Extension задайте произвольное имя (например my_server)

    ![](./images/hui_5.png)

    1. На вкладке Listen задайте порт (можно оставить по умолчанию)

    2. На вкладке TLS задайте следующие параметры:

      • tls.type = tls
      • cert: введите следующий ключ, используя параметры своего сервера

      /var/lib/docker/volumes/nginx_secrets/_data/live/hui{Server_ID_from_Invapi}.hostkey.in/fullchain.pem
      - key: введите следующий ключ, используя параметры своего сервера

      /var/lib/docker/volumes/nginx_secrets/_data/live/hui{Server_ID_from_Invapi}.hostkey.in/privkey.pem

    3. Нажмите на кнопку Add Config Item и выберите ACL. На появившейся вкладке ACL задайте следующие параметры:

      • acl.inline = reject(all, udp/443)

    4. Нажмите кнопку Save и переведи переключатель в положение Enable

    Теперь Hysteria2 успешно запущена, можно добавлять пользователей.

    Настройка конфигураций пользователей¶

    Чтобы предоставить доступ пользователям необходимо добавить пользователя с помощью меню Account Manage. Перейдите в меню и нажмите кнопку Add

    Настройте параметры пользователя (Логин, пароль,объем траффика, период доступа) и нажмите кнопку Confirm

    Теперь вы можете предоставить пользователю доступ. Для этого на главной странице нажмите Node URL или Node QR Code для более удобного подключения с мобильного устройства.

    Настройка доступа пользователей с использованием клиента Hiddify¶

    Внимание

    Если у вас происходит соединение с сервером, но не открываются некоторые сайты, проверьте настройки клиента. Он должен работать в режиме Системный прокси.

    На Смартфоне (Android и iOS)¶

    Лучший способ — использовать современные клиенты с поддержкой Hysteria 2.

    Android:

    Приложение: Hiddify (рекомендуется) или NekoBox.

    Инструкция:

    • Установите приложение из Google Play или GitHub.
    • Скопируйте ссылку hy2:// из панели H-UI.
    • Откройте приложение — оно часто само предложит добавить скопированный профиль (или нажмите Import from Clipboard)
    • Нажмите большую кнопку подключения.

    iOS (iPhone/iPad):

    Приложение: Hiddify (бесплатно), Shadowrocket (платно, универсальный комбайн) или Streisand.

    Инструкция:

    • Скопируйте ссылку из панели.

    • В Hiddify просто нажмите Добавить профиль из буфера. В Shadowrocket: при открытии он сам определит ссылку в буфере обмена и предложит добавить сервер.

    • Включите переключатель соединения.

    На Компьютере (Windows, macOS, Linux)¶

    • Скачайте Hiddify Next для вашей ОС.

    • Запустите, нажмите + (New Profile) и выберите Import from Clipboard.

    • Нажмите кнопку «Connect». Это решение автоматически настроит системный прокси.

    Информация

    Для корректной работы в Параметрах конфигурации необходимо включить

    • Enable DNS Routing
    • Strict Route
