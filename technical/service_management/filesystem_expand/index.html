    Как расширить файловую систему¶

    Внимание

    Перед изменением размера диска всегда необходимо сделать резервное копирование всех важных данных, хранящихся на этом диске. Изменение размера диска может привести к потере данных или их повреждению.

    Linux¶

    Информация

    GrowPart — инструмент командной строки Linux , используемый для расширения раздела в таблице разделов. Эта команда предоставляется пакетом облачных утилит.

    В нашем примере диск будет расширен на 10 Гб. Для этого необходимо выполнить следующий алгоритм действий:

    1. Проверка наличия свободного места на жестком диске с помощью команды lsblk. Если свободного места нет, то необходимо создать новый раздел, используя средства дистрибутива Linux.

      [root@45839 ~]# lsblk
NAME MAJ:MIN RM SIZE RO TYPE MOUNTPOINT
sda 8:0 0 40G 0 disk
├─sda1 8:1 0 1023M 0 part /boot
└─sda2 8:2 0 29G 0 part
├─vg45839-swap 253:0 0 3.7G 0 lvm [SWAP]
└─vg45839-root 253:1 0 25.3G 0 lvm /
sr0 11:0 1 1024M 0 rom

      В нашем примере корневая файловая система на физическом томе /dev/sda2. Объем диска - 30Гб.

    2. Применить утилиту growpart для расширения раздела, например, /dev/sda2:

      lsblk
growpart /dev/sda2
pvresize /dev/sda2
vgs
lvextend -r -l +100%FREE /dev/vg45839/root

    Windows¶

    1. Перейти в диспетчер дисков (сочетание клавиш Win+X) и выбрать пункт Управление дисками.

    2. Найти том, который необходимо расширить и нажать на его название правой кнопкой мыши, выбрать пункт Свойства.

    3. Во вкладке Объем нажать на кнопку Дополнительно, затем выбрать Расширить объем.

    4. Выполнить шаги в соответствии с инструкциями мастера и выбрать свободное пространство на жестком диске, чтобы добавить его к текущему тому.

    5. Нажать кнопку Готово и дождаться завершения операции.
